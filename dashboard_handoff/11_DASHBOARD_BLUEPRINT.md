## Component Inventory

### metric_card
```python
metric_card(label="ROC-AUC", value=0.9817, delta=0.002, help_text="Area under ROC curve")
```

### risk_indicator
```python
risk_indicator(score=0.75, label="High Risk")
```

### risk_bar
```python
risk_bar(score=0.75, height=20, width=200)
```

### Plotly Charts
- Line charts (temporal validation, robustness)
- Bar charts (feature importance, ablation)
- Heatmaps (confusion matrix, anomaly heatmap)
- Donut charts (risk distribution)
- Scatter plots (anomaly vs risk)

### Folium Maps
- CircleMarker (mine risk markers)
- HeatMap (risk density)
- Choropleth (province risk)
- Marker (FGR offices, border posts)
